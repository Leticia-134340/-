Routing Protocol
Roult: 경로
Router: 경로 안내 장치
Routing: 경로 안내 작업
Protocol: 통신위한 프로그램

정적
-static

-default

동적

AS
- IGP
--DISTANCE VECTOR: RIPv2

--LINK-STATE: OSPF

-EGP: BGP


AS(Autonomous System): 자치 시스템(관리자에 의해 관리되는 라우터의 집합/집단)
IGP(Interior Gateway Protocol): AS 내에서 연결(좁음)
EGP(Exterior Gateway Protocol): AS 밖에서 연결(넓음)
e.g) 각 통신사 내에서 서로 이어주는 건 IGP, 다른 통신사까지 해서 바깥까지 잇는 건 EGP

Distance Vector: Ripv2
전체 네트워크 학습 → 최적 경로 산출

미들웨어: hardware와 Application 사이에서 동작하게 만드는 거

ware: 제품
hardware: .CPU, RAM, DISK...부품
software: hardware 자원을 이용해서 특정 목적달성 위해 사용되는 컴퓨터 프로그램

System Software(Operating System): hardware 자원을 관리/ Application 관리/ file 관리

Application: 사용자 요구사항을 만족하는 프로그램
